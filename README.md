## Problem Statement
Traffic rule violations are increasing day by day, causing accidents and loss of lives.
Manual monitoring is inefficient and time-consuming.

## Solution
This project is an AI-based Smart Traffic Violation Detection system.
It detects traffic violations and displays alerts using a simple web interface.

## Features
- Detects traffic violations
- Simple and user-friendly UI
- Frontend + Backend based project
- Real-time alert message

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI/ML: Basic rule-based / AI logic
- Tools: VS Code, GitHub
